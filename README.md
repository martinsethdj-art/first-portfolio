# First Portfolio Project

This is my first frontend portfolio project built as part of the **FrontEnd Now** course.

The goal of this project was to practice:
- HTML structure
- CSS styling
- Flexbox layouts
- Mobile responsiveness
- Debugging layout issues

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Responsive Design

---

## Features
- Responsive header layout
- Search section with input and icon
- FAQ-style card grid
- Clean, organized project structure

---

- How to structure and clean up a GitHub repository properly
